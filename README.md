# octo-website

Octo 官网 — 现代化、国际化重写版。

## 关于 Octo

Octo 是一个开源 AI 原生团队协作平台，核心命题：**让 AI Agent 和人类真正共事**。

官网旧版：https://imocto.cn

## 技术栈（规划）

- **框架：** Next.js 15 (App Router)
- **样式：** Tailwind CSS + shadcn/ui
- **部署：** Vercel / 自托管
- **国际化：** next-intl (中/英双语)

## 开发

```bash
pnpm install
pnpm dev
```
